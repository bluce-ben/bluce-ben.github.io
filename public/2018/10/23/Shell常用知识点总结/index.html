4、比较两个字符串的大小

< 小于，在ASCII字母顺序下，如：
if [[ "$a" < "$b" ]]
if [ "$a" \< "$b" ]
< 大于，在ASCII字母顺序下，如：
if [[ "$a" > "$b" ]]
if [ "$a" \> "$b" ]
注意：在[]结构中”>”需要被转义。
