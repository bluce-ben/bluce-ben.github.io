引用

引用：>

世界上失败的人只有一种，那就是在成功路上放弃的人。
引用的引用：>>

世界上失败的人只有一种，那就是在成功路上放弃的人。

（注：> 和文本之间要保留一个字符的空格。）
注意：引用的区块内也可以使用其他的 Markdown 语法，包括标题、列表、代码区块等

粗体和斜体

使用星号（*）和底线（_）作为标记强调字词的符号，被（*）或（_）包围的字词会被转成用 <em> 标签包围，用两个（*）或（_）包起来的话，则会被转成 <strong>。
如果要在文字前后直接插入普通的星号或底线，你可以用反斜线进行转义。
（注：如果你的（\）和（_）两边都有空白的话，它们就只会被当成普通的符号。）*

代码引用

如果要标记一小段行内代码，你可以用反引号把它包起来（`）；
如果要标记一大段代码，可使用多个反引号来开启和结束代码区段：（```代码区段```）

链接

行内式链接语法：
[链接文字](图片地址 "title")
此处图片地址可使用绝对路径，也可使用相对路径。
（注：参考式链接此处不做说明。）
自动链接：
Markdown 支持以比较简短的自动链接形式来处理网址和电子邮件信箱，只要是用尖括号包起来， Markdown 就会自动把它转成链接。一般网址的链接文字就和链接地址一样，例如：
<http://example.com/>
Markdown 会转为：
<a href="http://example.com/">http://example.com/</a>

图片

行内式图片语法：
![Alt text](img "title")
（注：参考式图片此处不做说明。）
